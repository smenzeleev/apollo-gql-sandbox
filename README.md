# Apollo-gql-sandbox
> My sandbox for playing/learning with Apollo graphql client/server libs.

## Dependencies

- lerna
- yarn

## Start

In Root:

- Install Lerna if have not installed globally:
  ```yarn install```
- Install packages deps:
    - `yarn install`
    - `lerna bootstrap`
- Start front-end/back-end servers with yarn start
